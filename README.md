# 🏆 ZATCA Invoice Generator - مولد الفواتير الضريبية

## الفكرة

منصة SaaS عربية تولّد فواتير إلكترونية متوافقة مع هيئة الزكاة والضريبة (ZATCA/FATOORA) في ثوانٍ.

## المشكلة

- **1.7 مليون شركة** صغيرة ومتوسطة في السعودية ملزمة بفواتير إلكترونية
- الحلول الموجودة معقدة وغالية ($500+/شهر)
- الشركات الصغيرة تستخدم Excel يدوياً وتخربط
- الغرامات على عدم الامتثال حقيقية

## الحل

منصة بسيطة تولّد فواتير ZATCA-compliant بـ:
- QR Code متوافق مع تطبيق ZATCA
- حساب تلقائي لضريبة القيمة المضافة (15%)
- أرشيف فواتير + تصدير PDF/Excel
- إدارة عملاء بسيطة
- تقارير شهرية
